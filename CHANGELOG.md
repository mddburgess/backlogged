# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project
adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] — 2021-02-19

### Added

- Added support for adding and removing titles from a backlog.
- Added a navigation header to the UI.
- Added code linting and coverage checks to build.

### Changed

- Converted frontend code from JavaScript to TypeScript.

## [0.1.0] — 2020-12-05

### Added

- Added support for listing, adding, and updating library titles.
